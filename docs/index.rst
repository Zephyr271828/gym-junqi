Welcome to Gym Junqi's documentation!
=======================================

Gym Junqi is a reinforcement learning environment of the Junqi (Chinese Chess) game.
The project is built on top of a popular reinforcement learning framework called
`OpenAI Gym <https://gym.openai.com/>`_. Please check out this documentation page and our
`Github repository <https://github.com/Zephyr271828/gym-junqi>`_ for more information!
